# Vending Machine Visualizer

3D interactive visualizer for vending machine construction planning.

**Features:**
- Interactive 3D model with full component visibility controls
- Dimensions: 24"W × 60"H × 24"D
- Grid: 3×5 = 15 product slots
- Components view for plywood panel planning
- Modular codebase for easy maintenance

**Files:**
- `config.js` - All dimensions and parameters
- `materials.js` - Three.js materials and textures
- `utils.js` - Helper functions
- `components.js` - 3D component factories
- `ui.js` - UI controls and event handlers
- `main.js` - Application initialization

**Usage:** Open index.html in a browser
